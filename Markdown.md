# 제목
- h1 ~ h6
- `#`으로 작성

# 목록
- 순서가 없는 목록: `-`, `*`, `+`로 작성
- 순서가 있는 목록: `1.`, `2.` 등 숫자로 작성

# 글꼴
- *기울임*: `*글자*`, `_글자_`, *ctrl + i*
- **굵게**: `**글자**`, `__글자__`, **ctrl + b** (단축키는 타이포라)
- ~~취소선~~: `~~취소선~~`
- 밑줄: ctrl + u (타이포라)

# 코드
- 한 줄 코드: `인라인 코드` (백틱으로 감싸기)
- 여러 줄 코드: 블록 코드 ```python (백틱 3번 입력하고 코드의 종류 작성, 백틱 3번 감싸기)
